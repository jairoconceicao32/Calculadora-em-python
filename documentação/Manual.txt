Essa calculadora de momento é toda controlada por letras, ou seja, cada letra representa uma função do menu.
1. Calcular C:
Nessa opção é apresentado um campo, ao qual o usuário precisa informar um valor
O segundo campo é o menu das operações matemáticas, agora controlado pelos cinais das operações!
O terceiro campo é para informar o valor a ser calculado.
Após isso, vai exibir o resultado da operação feita na tela.
2. Histórico H:
Nessa opção é nos apresentada opções relacionadas ao histórico. São elas:
2.1. Ver histórico V:
Nos permite ver o histórico das operações feitas bem como os resultados.
2.2. Copiar o histórico C:
Nos permite copiar o histórico para a área de transferência.
2.3. Limpar histórico L:
Limpa o histórico, apagando ele por completo.
2.4. Retornar ao menu principal R:
Volta ao menu principal.
3. Sair:
Fecha a calculadora.
